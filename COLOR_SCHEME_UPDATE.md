# Color Scheme Update - Green to Red

## Summary
Successfully updated the entire Lummus website from emerald/green color scheme to red, matching the actual Lummus branding at https://www.lummus.com/

## Changes Made

### Color Replacements:
- `emerald-50` → `red-50` (light backgrounds)
- `emerald-100` → `red-100` (lighter accents)
- `emerald-600` → `red-600` (medium red)
- `emerald-700` → `red-700` (primary brand color)
- `emerald-800` → `red-800` (darker hover states)
- `emerald-400` → `red-400` (footer accents)
